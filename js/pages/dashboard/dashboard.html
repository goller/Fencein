Welcome to Fence-in!

The fencing tournament check-in app.

Checking in fencers

  • Click the Check In link at the top
  • Select the tournament from the Select Tournament list
  • Select the person to check in, the table can be sorted by clicking on the column header
  • Verify that the information is correct
  • Select any events that they would like to participate in
  • If they would like to borrow equipment click equipment checkout button
  • Take their money

    • To get started set up a tournament using the Tournaments link in the navigation bar.

    When you are ready to check fencers into a tournament events click on the checkin link and select the corrent tournament.
